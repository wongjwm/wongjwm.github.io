The Easy Animator

The Easy Animator is a Swing-based GUI application that animates given text commands into an animation. It also features additional functionality, such as the option to loop the animation, adjust the speed, select and deselect which shapes appear in the animation, and the option to export and save the animation in a 'SVG' format. Built using Java and the Swing framework. Created for Object-Oriented Design final project.
